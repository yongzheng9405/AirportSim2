1. In airport folder, Airplane.cpp and Airplane.h includes functions related to airplanes.
2. Airport.cpp and Airport.h includes functions related to airports.
2. Event.cpp and Event.h includes functions related to events.
4. To compile, cd into src folder and use command:
	g++ -std=c++0x *.cpp -o out
   to run it, use command:
	./out
5. Folder AirportGUI includes our code for GUI written in C++. You can run AirportGUI.sln
   to display the GUI in Visual Studio!!!! If you did not install Visual Studio, you can
   refer to our video or ask us for further demonstration.
